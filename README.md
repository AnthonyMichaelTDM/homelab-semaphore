# Homelab Semaphore

A repository containing ansible playbooks, shell scripts, and other resources for managing my homelab through Semaphore.

## Structure

### Ansible

- `playbooks/`: Contains Ansible playbooks for various tasks
- `roles/`: Contains Ansible roles
- `inventory/`: Contains inventory files for different environments
- `collections/`: Contains Ansible collections

### Shell Scripts

- `scripts/`: Contains shell scripts for various tasks
